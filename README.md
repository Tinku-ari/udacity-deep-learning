## Install Directions

```
$ conda env create
$ pip install -U pip
$ curl "https://storage.googleapis.com/tensorflow/mac/tensorflow-0.6.0-py2-none-any.whl" -o "tensorflow-0.6.0-py2-none-any.whl"
$ git clone git@github.com:tensorflow/tensorflow.git tensorflow_source
$ cd tensorflow_source/tensorflow/examples/udacity/
$ jupyter-notebook
```
